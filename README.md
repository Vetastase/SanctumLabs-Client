## Client/Frontend

<br>

### Routes

- / - Transition(Landing Page)
- /home - Home Page
- /signup - Signup Form
- /login - Login Form
- /profile - Your Profile
- /cards - Cards List
- /cards/create - Adding Cards
- /cards/:cardId - Card Details
- /cards/edit/:cardId - Edit information or delete card

<br>
